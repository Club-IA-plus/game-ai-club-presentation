# Jeu Phaser 3 - Game AI Club

Projet de jeu 2D utilisant Phaser 3.

## Installation et utilisation

### Commandes disponibles

- `make dev-build` - Construit l'image Docker
- `make dev-run` - Lance le conteneur en mode développement
- `make dev-kill` - Arrête et supprime le conteneur

http://51.178.80.14:8080

### Démarrage rapide

```bash
make dev-build
make dev-run
```

Le jeu sera accessible sur http://localhost:8080

### Structure du projet

- `index.html` - Page principale
- `js/game.js` - Code du jeu Phaser 3
- `Dockerfile` - Configuration Docker
- `docker-compose.yml` - Configuration Docker Compose
- `Makefile` - Commandes de développement
